# Soma de Matrizes em Java

## Descrição
Este projeto consiste em um programa desenvolvido em Java que realiza a leitura de duas matrizes 2x2 informadas pelo usuário.

Após a entrada dos dados, o programa executa a seguinte operação:
- Soma os valores correspondentes das duas matrizes

O resultado da soma é exibido ao usuário em formato de matriz.

A interação com o usuário é feita através de janelas utilizando a classe `JOptionPane`.

## Como executar o programa

1. Clone o repositório ou baixe os arquivos
2. Abra o projeto em uma IDE Java (ex: IntelliJ IDEA ou Eclipse)
3. Compile o projeto
4. Execute a classe principal
5. Insira os valores das matrizes nas janelas que irão aparecer na tela

## Exemplo de uso

### Entrada:

Matriz A: 1 2
          3 4
          
Matriz B: 6 8
         10 12

## Tecnologias utilizadas
- Java
- JOptionPane (interface gráfica simples)
